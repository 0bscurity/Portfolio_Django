HashTrackr

HashTrackr is a lightweight and user-friendly python application that monitors specified websites for content changes based on hashes of its content. Utilizing Tkinter for its graphical user interface, HashTrackr provides an easy-to-use platform for users to input a URL and receive real-time updates on any modifications to the website content. HashTrackr continuously checks the specified URL for content changes at regular intervals, notifying users of any updates. Plans to add a text notification option in the works.

This Website

I built this portfolio as a full-stack Django application to consolidate and showcase my work. It is hosted on a Vultr virtual machine, utilizing Nginx as a web server and reverse proxy, and leveraging Docker for seamless containerization. During the development process, I encountered several challenges, including the integration of Docker Compose and Nginx configurations, as well as the acquisition and maintenance of SSL certificates for secure HTTPS access. Despite these obstacles, the experience has been highly educational and rewarding, and I am eager to expand my skillset by hosting additional applications of my own creation.
